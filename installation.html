Installation¶

Stable release¶

To install Cloud-BIDS-Layout, run this command in your terminal:

$ pip install cloud_bids_layout

This is the preferred method to install Cloud-BIDS-Layout, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for Cloud-BIDS-Layout can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/nrdg/cloud_bids_layout

Or download the tarball:

$ curl -OJL https://github.com/nrdg/cloud_bids_layout/tarball/master

Once you have a copy of the source, you can install it with:

$ python setup.py install
